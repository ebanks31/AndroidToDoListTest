Test Application for ToDoList Android application.

This application test application for the To Do List Android application.
The user can create a new title for the new list and add items to the list.
These classes uses to Robotium framework in order to test the application.

With the Robotium framework, you can imitate the users key presses in the code and check
if the key presses return the correct result.

Robotium Link is located here: https://code.google.com/p/robotium/
